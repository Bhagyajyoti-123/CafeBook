# ☕ CafeBook

CafeBook is a full-stack web application for managing café bookings, user wallets, and memberships.  
It demonstrates authentication, REST API integration, and dynamic frontend-backend interaction.

---

## 🚀 Features

- 🔐 User Authentication (Signup / Login)
- 💰 Wallet Recharge System
- 👑 Membership Activation / Deactivation
- 📊 Real-Time Balance Tracking
- 🏪 Café Booking with Transaction History
- 🚪 Secure Logout & Session Handling

---

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- REST API (Node.js based backend running on localhost:8081)

---

## 📌 How It Works

1. Users create an account and log in.
2. UserId is stored in localStorage for session management.
3. Users can:
   - Recharge wallet
   - Activate membership
   - Book cafés
   - View booking history
4. All data is fetched dynamically from backend APIs.
